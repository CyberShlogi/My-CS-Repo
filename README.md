# National Institute of Standards and Technology
**The NIST Cybersecurity Frameworks (CSF)**

A voluntary framework that consists of standards, guidelines, and best practice to manage cybersecurity risk.

**NIST CFS Components**

>Core

>Tiers

>Profiles

**Five functions of the CSF Core**

>Identity

>Protect

>Detect

>Respond

>Recover

**Tiers**

The CFS consists of four tiers ranging from tier 1 which is passive up to tier 4 which is adaptive.

**Profiles**

Profiles provides useful insights of the current state of security pratice.
